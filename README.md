Team Task Manager

A simple and responsive Team Task Management web application built using HTML, CSS, and JavaScript. This project allows users to create projects, manage tasks, track progress, and monitor overdue work through a clean dashboard interface.

Features
User Login with Admin and Member roles
Create and manage projects
Add tasks with deadlines and status
Mark tasks as completed
Dashboard to track:
Total tasks
Completed tasks
Overdue tasks
Simple and responsive UI
Technologies Usedaa
HTML
CSS
JavaScript
